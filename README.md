# SufX Portfolio

This repository contains the source code for my personal developer portfolio website. It is designed as a clean, responsive, and lightweight web application built to showcase technical projects and professional experience.

## About the Project

The application serves as a centralized digital workspace, structuring and displaying my work across web development and integrated machine learning applications. The architecture prioritizes fast load times, a minimal user interface, and seamless cross-device compatibility.

## Built With

* **Framework:** React
* **Build Tool:** Vite
* **Styling:** Tailwind CSS

## Key Features

* **Responsive Design:** Fluid and optimized layouts for mobile, tablet, and desktop viewing.
* **Project Showcase:** Structured display of featured applications, including technology stacks and source code links.
* **Modern UI:** Clean and maintainable component-based styling using Tailwind CSS.
